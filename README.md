# Angular2 Starter Kit

A repo to quickly scaffold a basic Angular2 project.

### Includes
* NG2 Core Components
* Bootstrap
* SystemJS
* Typescript
* Zone.js
* rxjs
* Lite Server
* BrowserSync

### Usage
1. Run `npm install`
2. Run `npm start`
3. Start Developing

*Lite server should serve on `http://localhost:3000`*
